Key Features

  • Multi-model training & automatic best-model selection
  • Aspect-Based Sentiment Analysis (7 categories)
  • RAG-style evidence retrieval from training corpus
  • Word-level positive/negative signal highlighting
  • Confidence scoring via sigmoid on decision boundary
  • Responsive professional dark-mode UI
📊

Dataset & Methodology

Trained on the IMDB Movie Reviews benchmark — 50 000 labelled reviews. Pre-processed with NLTK (lowercase, HTML stripping, stop-word removal, Porter stemming) then vectorised with TF-IDF (15 k features, 1–2 gram range). RAG retrieval uses Jaccard bag-of-words overlap — no heavy embedding dependencies.
